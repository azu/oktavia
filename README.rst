Oktavia
=======

Oktavia is a full text search engine for JavaScript environment. It uses FM-Index.

Command
-------

Generate JS code
~~~~~~~~~~~~~~~~

.. code-block:: bash

   # for web envrionment
   $ make release_web

   # for node.js
   $ make release_node

Run test
~~~~~~~~

.. code-block:: bash

   $ make test
